# Heart Disease Predictor (Innovative IT Project)

### Overview
This project is a Data Science application built to assist medical professionals in identifying heart disease risks. It uses a **Random Forest Classifier** trained on the Cleveland Heart Disease dataset.

### Key Features
* **Machine Learning:** Real-time prediction based on patient metrics.
* **Interactive UI:** Built using Streamlit for a seamless user experience.
* **Data Driven:** Uses industry-standard health datasets for training.

### How to Run
1. Clone this repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
